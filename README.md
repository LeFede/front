# Volpe Template

#### Get it working:

```sh
git clone https://github.com/LeFede/front
cd front
pnpm i
npm run dev
```

---

#### Testing:

```sh
npm run cypress:run
npm run cypress:open
```

```sh
npm test # once
npm run testw # watch
```
